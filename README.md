# Heart Health Data Analysis

## Overview
This project analyzes a heart health dataset using Microsoft Excel to identify key risk factors and patterns associated with heart disease.

## Objective
To perform exploratory data analysis (EDA) and generate insights that help understand the relationship between health indicators and heart disease risk.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning
- Charts & Dashboard

## Key Analysis Performed
- Age distribution analysis
- Cholesterol level comparison
- Blood pressure trend analysis
- Risk factor identification using categorical data
- Summary statistics calculation

## Insights
- Identified key variables influencing heart disease risk
- Observed patterns between age, cholesterol, and health conditions
- Created dashboard for clear visualization of results
